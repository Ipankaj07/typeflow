# ⌨️ TypeFlow

TypeFlow is a premium, minimalist typing test application designed for focus and speed. Master your keystrokes with a beautifully crafted interface, real-time feedback, and a suite of high-end aesthetic themes.

![TypeFlow Preview](/og-image.png)

## ✨ Features

- **🎯 Precision Tracking**: Real-time WPM, accuracy, and raw speed metrics.
- **🎨 Premium Themes**: 5 curated aesthetic themes (Light Flow, Midnight, Sakura, Obsidian, Terminal).
- **🔊 Auditory Experience**: Mechanical keyboard sounds (click/thock) and auditory feedback for errors.
- **📈 Performance Analytics**: Dynamic charts visualizing your progress over the duration of the test.
- **🛠️ Custom Tests**: Paste your own paragraphs or choose from curated collections.
- **📱 Responsive Design**: Fully optimized for desktops and tablets.

## 🚀 Getting Started

### Prerequisites
- Node.js 18.x or later
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/pankaj/typeflow.git
   cd typeflow
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to experience TypeFlow.

## 🛠️ Technology Stack

- **Framework**: [Next.js 15+](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Audio**: [Web Audio API](https://developer.mozilla.org/en-US/docs/Web/API/Web_Audio_API)
- **Visualization**: [Recharts](https://recharts.org/)
